# badge-generator
Image badge generator

```
Usage: badge-generator [options]

Options:
  -v, --version            output the version number
  -i, --input [path]       Input image path
  -o, --output [path]      Output image path
  -t, --text [string]      Text to print, you can add \n for line breaks
  -w, --width [integer]    Width of the output image (default: same as input)
  -h, --height [integer]   Height of the output image (default: same as input)
  -p, --padding [integer]  Padding in percentage of the output size (default: 10)
  -c, --color [string]     Text color (default: #ffffff)
  -f, --font [string]      Font to use (default: Arial)
  -h, --help               output usage information
```
